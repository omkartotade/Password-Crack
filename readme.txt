1. run run.c to produce all possible hashes for 4 letter passwords.



2. open crack.c file and provide the hash to crack to the char array ss[]. Execute crack.c.

run.c is a program that uses the MD5 hashing technique to create hashes of all possible 4-character password (excluding special characters) and stores it in a file. 

run.c is a program that accepts any hash value and cracks the 4-character password for it.
